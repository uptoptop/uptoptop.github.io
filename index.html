📋 Hướng dẫn cài đặt

  1. Mở Kodi → Settings (⚙️) → File Manager
  2. Chọn Add source → nhập địa chỉ: https://uptoptop.github.io
  3. Đặt tên nguồn (ví dụ: Fuck.You.BI>) → nhấn OK
  4. Quay lại → Add-ons → Install from zip file → chọn nguồn Zinzuno
  5. Chọn repo_vietmediaf_fork-0.0.2.zip → chờ thông báo cài đặt thành công
  6. Vào Install from repository → VIETMEDIAF_FORK → Video add-ons → VietmediaF → Install
  7. Addon sẽ tự động cập nhật khi có phiên bản mới 🎉
💡 Mẹo: Nếu bước 6 (Install from repository) thất bại, bạn có thể cài đặt trực tiếp bằng cách vào Install from zip file → chọn nguồn Fuck.You.BI → plugin.video.vietmediaF → chọn plugin.video.vietmediaF.zip.
